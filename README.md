# Movies app

A simple flutter application with firebase authentication to fetch:
<ul>
   <li> Trending Movies </li>
   <li> Top Rated Movies </li>
   <li> Now Playing Movies </li>
</ul>

They are fetched using api from tmdb (https://www.themoviedb.org/)

# Tech Stack

<ul>
   <li> Flutter </li>
   <li> Firebase </li>
</ul>
